What Can I Handle

Publishing & Design

HTML
웹 표준과 접근성을 준수하는 HTML. 정보 구조 설계로 웹사이트 퍼플리싱을 해오고 있습니다. 마크업 방식과 시멘틱 태그를 사용해 전체 레이아웃을 설계합니다. 웹 UI 디자인 작업 후 클래스 네임과 함께 HTML 정보 구조설계를 꼼꼼하게 작업한 후 퍼블리싱 작업을 진행합니다.
CSS
여러 CSS 속성을 이해하고 파악하여 HTML을 배치, 디자인 합니다. 새로운 CSS3 속성 또한 활용하여 좀 더 다양한 디자인을 할 수 있도록 연구하고 공부하고 있습니다.
JQUERY
네비게이션 , 탭 메뉴 , 아코디언등 웹 퍼블리싱에 필요한 제이쿼리를 플러그인을 사용하지 않고 순수 제이쿼리로 제작이 가능하며, 상황에 따라 여러 플러그인을 활용하여 제작하고 있습니다.
MOBILE WEB APP
웹페이지를 모바일로 변환 작업의 경험을 가지고 있습니다. ui/ux 측면에서 디자인 뿐만 아니라 모바일 사용자의 최적화된 환경을 위해 컨텐츠를 효율적으로 배치하는 것을 우선으로 생각하며 제작합니다.
RESPONSIVE WEB
반응형 웹 제작에 있어 Position, Float, Media Query를 활용한 반응형 제작 경험이 있고 또한 Flex와 Grid 방식을 사용하여 반응형 웹 제작을 하고 있습니다.
TEXT Editor
가장 필수적으로 Visual Studio Code를 잘 이해하며 사용하고 있으며 필요에 따라 Editplus등의 에디터를 병행하여 사용하고 있습니다.
GIT / GITHUB
git을 이용하여 로컬 저장소에서 커밋하여 분산 버전 관리를 용이하게 하고 있고 GITHUB 레퍼지토리에 업로드 하며 활용하고 있습니다.
Adobe Photoshop
포토샵 프로그램을 활용하여 퀄리티 있는 웹디자인 소스 구축 및 디자인이 가능합니다. 또한 이미지에 대한 기본 이해도를 바탕으로 각 홈페이지 레이아웃에 최적화된 트렌디한 구성의 작업 진행이 가능합니다.
Video Tools
유튜브 채널을 위한 업로드 영상을 Finalcut 사용하여 편집 경험이 다수 있으며 촬영 프로젝트를 진행하며 컷편집, 음향 편집이 가능합니다.
